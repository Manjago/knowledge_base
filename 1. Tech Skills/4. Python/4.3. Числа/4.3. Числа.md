# Числа
## Операции с Integer
```python
>>> 2 + 3
5
>>> 3 - 2
1
>>> 2 * 3
6
>>> 3 / 2
1.5
>>> 3 ** 2
9
```
`**` - возведение в степень<br/>
_Eric Matthes, Python Crash Course, 2nd Edition, p.26_

## Float
В Python все числа с плавающей точкой - `float`<br/>
_Eric Matthes, Python Crash Course, 2nd Edition, p.26_

## Остаток от деления

```python
>>> 4 % 3
1
>>> 6 % 3
0
```
